A Code Repo for a Classic Snake Game dveloped using Advanced Java Swing. Implemented real-time movement, collision detection, and dynamic score tracking with interactive GUI (Graphical User Interface).
To Execute this Code: 
Open the terminal inside the project folder that contains both files (SnakeGame.java and GameArea.java) and run the following commands: 
1)  java src/SnakeGame.java src/GameArea.java // IntelliJ Idea(Command)
2)  java src/SnakeGame 
This will compile and start the game.
